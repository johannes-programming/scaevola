scaevola
Introduction

This project contains the Scaevola class which can be used as a base class to utilize its preset right-handed magic methods: __ge__, __gt__, __radd__, __rand__, __rdivmod__, __rfloordiv__, __rlshift__, __rmatmul__, __rmod__, __rmul__, __ror__, __rpow__, __rrshift__, __rsub__, __rtruediv__, __rxor__.

Example

The __radd__ magic method is defined as below. The others follow the same pattern.

def __radd__(self: Self, other: Any) -> Any:
    "This magic method implements other+self."
    return type(self)(other) + self
